Description of the Vec class

The Vec class implements one-dimensional dense vectors in GroovySci.



Constructors

  1. Vec(int len): Creates a vector of size len initialized to zeros

v = new Vec(100) // creates a vector with 100 elements initialized to zero values.



2. Vec(double []da): Creates a vector initialized with the da array

da = new double[20]

da[3]=3

dav = new Vec(da, true)



Routines

linspace: Linearly spaced vector.

linspace(x1, x2) generates a row vector of 100 linearly equally spaced points between x1 and x2.

linspace(x1, x2, N) generates N points between x1 and x2.



logspace: Logarithmically spaced vector.

logspace(x1, x2, N, base) generates a row vector of N logarithmically equally spaced points between x1 and x2.

vones(int N:) : generates a N-sized vector of ones

vzeros(int N) : generates a N-sized vector of zeros

    vfill(int N, double value): generates a N-sized vector filled with value

    abs, min, max, sin, cos, tan, asin, acos, atan, sinh, cosh, tanh, ceil, floor, sqrt, log, exp: the corresponding routines
