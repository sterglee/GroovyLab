Description of the Matrix class

The Matrix class implements zero-indexed two-dimensional dense matrices in GroovySci. That matrices are mostly based on the JAMA library.



Constructors



Matrix(rows: Integer, cols: Integer) // Creates a Matrix of size rows, cols initialized to zeros

m=new Matrix(2, 3)

Construction by specifying the initial elements, e.g.

m = new Matrix([[4.5, 5.6, -4], [ 4.5, 3, -3.4]] as double[][])



// construct a Matrix by copying the array values

Matrix(double [][] da ) // Creates a Matrix initialized with the da array

dd = new double[2][4]

dd[1][1]=11

mdd = new Matrix(dd)



Construction with filling an initial value,

Matrix(int N, int M, double value), e.g.

m = new Matrix(6,8, 6.8)



Resamples a Matrix every n rows and every m cols,

Matrix resample(int n, int m), e.g.

m = rand(20, 30)

mr = m.resample(2,3)





Matlab-like submatrix row and column extraction operation: (fromR:incR:toR, fromC:incC:toC),

Matrix [int fromR, int incR, int toR, int fromC, int incC, int toC], e.g.

m = rand(20, 30)

msubm = m[2, 2, 10, 4, 3, 10]



Matlab-like submatrix row extraction operation: (fromR:incR:toR, :),

Matrix[int fromR, int incR, int toR, “:”]), e.g.

m = rand(20, 30)

msubm = m[2, 4, 10, ":"]



Matlab-like submatrix column extraction operation: (:, fromC:incC:toC),

Matrix submc(int fromC, int incC, int toC), e.g.

m = rand(20, 30)

msubmc = m.submc(2, 2, 10)
