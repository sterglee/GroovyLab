Gaussian quadrature – General weights

reccof

Evaluate the coefficients c_1, c_2 in the recursion

C_{k+1}(x) = x C_{k}(x) – c_k C_{k-1}(x) (k=1,2)

for the Chebyshev polynomials of the second kind which are orthogonal over [-1, 1] with respect to the weight function w(x) = (1-x^2)^{1/2}



Example: Can be run with F9 within the GroovyLab Editor (Ctrl-E to open the editor)
