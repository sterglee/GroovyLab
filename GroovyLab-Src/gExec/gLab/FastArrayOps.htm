Fast operations on 2-Ds arrays (i.e. double [][])

Although gLab classes provide Matlab-like flexible matrices, they do not operate as fast as plain 2-D arrays, especially for large sizes. gLab offers extensive support to work directly on 2-D arrays. The following routines are provided:

  Basic Mathematical Operations

1. Matrix multiplication

Let us create a trial 2-D array:
        a = Ones(9,9)
Muliply it with itself i.e.
         aa = a * a
This method invokes a Java based matrix multiplication routine

2. Addition and subtraction operations are all overloaded to operate both with double [][] arrays and numbers

e.g. ff = a*8.8+6.7*a
